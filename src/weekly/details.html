Week 1: Introduction to HTML

Starts on: 2024-01-15

Description & Notes

This week introduces the fundamentals of HTML, including its structure, common tags, and best practices for creating web pages. Students will learn how to create headings, paragraphs, links, images, and lists. By the end of the week, students should be able to build a simple static webpage using HTML.
